¿Qué es Jekyll?

Ir al grano

Principalmente se tratan de blogs o páginas webs en las que no es necesario tener una base de datos. Por ejemplo, si una persona quiere una página web para su restaurante, peluquería, taller o cualquier otro negocio en la que simplemente se usa para tener una “imagen de empresa” en internet.
